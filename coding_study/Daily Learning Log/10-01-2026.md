The following code demonstrates how to refactor a standard timer into a **Promise-based function**, allowing you to use `await` for cleaner sequential execution as shown in the sources.

```
// A helper function that returns a Promise to handle delays
const setTimeoutPromise = (ms) => new Promise((resolve) => {
  setTimeout(resolve, ms);
});

async function drawLottoBalls(winBalls) {
  for (let i = 0; i < winBalls.length; i++) {
    // Execution pauses here for 1 second before moving to the next line
    await setTimeoutPromise(1000); 
    console.log(`Ball ${i + 1}: ${winBalls[i]}`);

  }
}
```

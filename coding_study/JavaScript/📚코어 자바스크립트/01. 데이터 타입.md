# 01 데이터 타입의 종류 #
## 기본형 - 원시형 (Primitive Type) ##
- 숫자 number
- 문자열 string
- 불리언 boolean
- null
- undefined
- 심볼
---
### 기본형 ###
- 할당 + 연산시 복제 됨
	- 값이 담긴 주솟값을 바로 복제
- 불변성 (immutability)
	- 개략적 

## 참조형 - Reference Type ##
- 객체 object
- 배열 array
- 함수 function
- 날짜 date
- 정규표현식 RegExp
- Map
- WeakMap
- Set
- WeakSet
---
### 참조형 ###
- 참조됨
	- 값이 담긴 주솟값 묶음을 가리키는 주솟값을 복제

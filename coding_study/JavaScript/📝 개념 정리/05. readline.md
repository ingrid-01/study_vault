⚠️ Node.js 환경에서 데이터를 받아오는 '기계적인 세팅'
	- JS 언어 자체의 핵심 문법은 아님!!!

## "택배 상자" ##
- `readline` 이란
	- **택배를 받아서 내용을 확인하는 과정**
		- `readline` 모듈: 집 현관문 (입구)
		- `rl.on('line, ...)`: 택배 기사님이 상장 하나를 건넴
			- '상자 하나 도착!' → `input` 이라는 바구니에 담김
		- `rl.on('close', ...)`: 기사님 → "배달 끝, 굿바잉" → 문 닫음
			- 문 닫힌 뒤, 바구니(`input`)를 열어 내용물(`str`) 꺼내서 사용(코딩)을 하는 것
## 코테에서의 대처법: "여기는 노터치" ##
- 이 코드가 계속 보인다면:
	- "위쪽은 그냥 문제 데이터를 받아오는 준비 과정이구나."
	- "내가 건드릴 곳은: `close` 안쪽!"

```js
const readline = require('readline');
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});

let input = [];

rl.on('line', function (line) {
    input = [line];
}).on('close',function(){
    str = input[0];
/* 여기까진 신경 끄고 밑에부터 건드리기*/

    console.log(str);
});
```

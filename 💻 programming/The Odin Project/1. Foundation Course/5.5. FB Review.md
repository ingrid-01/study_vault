# CSS Cheatsheet #
## 1. Style & Layout ##

**a) Colours** - text-color, backgroun-color, borders, outline etc. (RGB, HEX, HSL)
```css
p {
	color: darkred;
	background-color: #f0f0f0;
	border: 2px solid rgb(0, 128, 255);
	outline: 1px dashed red;
}
```

**b) Fonts** - Font family, Font size, Weight (boldness), Style (italics, normal), Line height & Letter Spacing etc. 
```css
body {
	font-family: 'Helvetic', sans-serif;
	font-size: 16px;
	font-weight: 400px; /* normal */
	font-style: italic; /* normal (default), oblique */
	line-height: 1.6;
	letter-spacing: 0.5px;
}
h1 {
	font: italic 16px Arial, sans-serif; /* shothand for style, size and family */
}
```

**c) Spacing** - margin, padding, gap
```css
h1 {
	margin-bottom: 20px; /* Space below heading */
}
.container {
	padding: 16px; /* Space inside the container box */
	gap: 1rem; /* Space btw rows and columns */
}

/* padding & border stay inside the defined width/height 
Use this as a boilerplate */
*,
*::before,
*::after {
	padding: 0;
	margin: 0;
	box-sizing: border-box;
}
```

**d) Positioning** - where elements appear on WP
- `position` - static (default), relative (to its container), absolute (rogue), fixed (to viewport), sticky (stays in its block)
- `top, bottom, left, right`: used with `position`; relative to viewport
- `z-index`: layer order (which comes first via stacking bigger the #, most top layer)

```css
/* Relative to its normal position */
.relative-box {
	position: relative;
	top: 10px;
	left: 20px;
}

/* Absolute positioning inside nearest relative parent */
.absolute-box {
	position: absolute;
	bottom: 0;
	right: 0;
}

/* Sticky header */
.header {
	position: sticky;
	top: 0;
	background: white;
	z-index: 100;
}
```
## 2. How CSS works ##
- applied:
	- Inline - `<div style="color: red;">`
	- Internal - `<style>` inside `<head>`
	- External - linked `.css` file
## 3. Box Model ##
`content` → text/image
`padding` → space inside border
`border` → edge of the box
`margin` → space outside the box

## 3. Key Properties ##

| Property          | Target    | Description                                                                                                                 |
| ----------------- | --------- | --------------------------------------------------------------------------------------------------------------------------- |
| `display: flex`   | container | Turns children into flex items; initiation                                                                                  |
| `flex-direction`  | container | `row` (default), `column`, `row-reverse`, `column-reverse`                                                                  |
| `justify-content` | container | Aligns items on **main** axis; `flex-start, flex-end, center, space-between, space-around`                                  |
| `align-items`     | container | Aligns items on **cross** axis; `flex-start, flex-end, center, baseline, stretch`                                           |
| `order`           | item      | default: 0; interger values (-2, -1, 0, 1, 2)                                                                               |
| `align-self`      | item      | Overrides `align-items` for a single item; same value as `align-items`                                                      |
| `flex-wrap`       | container | Allows items to wrap to next line; `nowrap; wrap; wrap-reverse`                                                             |
| `flex-flow`       | container | `flex-direction + flex-wrap`                                                                                                |
| `align-content`   | container | Space btw lines (when there's only 1 line, no effect); `flex-start, flex-end, center, space-bewteen, space-around, stretch` |
| `flex`            | item      | Sets size ratio (`flex: 1, 2, etc.);` ; `flex-grow + flex-shrink + flex-basis`                                              |
| `gap`             | container | Adds spacing btw items; `gap: 1rem`                                                                                         |


| `flex` value      | Meaning                        | Use Case                    |
| ----------------- | ------------------------------ | --------------------------- |
| Default           | -                              | Default: `flex: 0 1 auto`   |
| `flex: 1`         | Grow equally, no initial width | `flex: 1` = `flex: 1 1 0%`  |
| `flex: 2`         | Grow twice as fast             | One item larger than others |
| `flex: 0 0 auto`  | Fixed size based on a content  | Button, logos, etc.         |
| `flex: 0 0 100px` | Fixed width                    | sidebar, nav elements       |
| `flex: 0 1 auto`  | Shrinkable, based on content   | Responsive text containers  |
1. Equal Width Columns
	ALL items grow EQUALLY to fill space
	- equal width
	- good for grids, nav bars etc.
```css
.item {
	flex: 1;
}
```

2. One Item bigger than the rest
	Make one item grow more:
	- `item-1` takes twice as much space as `item-2`
```css
.item-1 {
	flex: 2;
}
.item-2 {
	flex: 1;
}
```

3. Fixed Width w No Grow
	- don't grow or shrink
	- Always 200px wide
```css
.item {
	flex: 0 0 200px;
}
```

4. Auto grow from content
	- Item size is based on content but can shrink if needed
```css
.item {
	flex: 0 1 auto;
}
```

# 3. JS Script #
```js
// Step 1: Get references to HTML elements
const numberInput = document.getElementById('numberInput');
const checkBtn = document.getElementById('checkBtn');
const result = document.getElementById('result');
```
- Getting Elements:
	- `document.getElementById()` finds HTML elements by their ID
	- We store references in variables so we can use them later
	- `const` means these references won't change

```js
// Step 2: Create the function that checks if a number is odd or even
function checkOddOrEven() {
    // Get the value from the input field
    const inputValue = numberInput.value;
    
    // Convert the input to a number
    const number = parseInt(inputValue);
    
    // Check if the input is valid (not empty and is a number)
    if (inputValue === '' || isNaN(number)) {
        // Display error message if input is invalid
        result.textContent = 'Please enter a valid number!';
        result.className = 'error'; // Add error styling
        return; // Exit the function early
    }
```
`if (inputValue === '' || isNaN(number))`:
- Input Validation:
	- Always check if user input is valid before using it
	- `isNaN()` means "is Not a Number" - returns true if it's not a valid number
	- `return;` exits the function early if input is bad

```js
    // Clear any previous styling classes
    result.className = '';
    
    // Use the modulo operator (%) to check if number is odd or even
    if (number % 2 === 0) {
        // If remainder is 0, the number is even
        result.textContent = `${number} is EVEN!`;
        result.className = 'even'; // Add even styling
    } else {
        // If remainder is not 0, the number is odd
        result.textContent = `${number} is ODD!`;
        result.className = 'odd'; // Add odd styling
    }
}

// Step 3: Add event listener to the button
checkBtn.addEventListener('click', checkOddOrEven);

// Step 4: Allow Enter key to trigger the check (bonus feature)
numberInput.addEventListener('keydown', function(event) {
    // Check if the pressed key is Enter (key code 13 or 'Enter')
    if (event.key === 'Enter') {
        checkOddOrEven(); // Call the same function as the button
    }
});
```

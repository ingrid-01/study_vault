## 지역 변수 Local Variable ##
- 함수 안에서 선언 + 함수 안에서만 사용
	- 지역 변수 선언 = 예약어 `var` + 변수 이름 지정
		- 변수 이름 앞에 `var` X = 전역 변수로 취급!
```js
var myVar = 100; // 전역 변수 선언
test();
document.write("myVar is " + myVar);

function test() {
	var myVar = 50; // 지역 변수 선언 
}
```
`test()` 함수 밖에 변수 `myVar` = 100
`test()` 함수 안에 변수 `myVar` = 50 → function안에 써놓은 var myVar는 지역 변수이기에 myVar는 100으로 표시됨!

- 전역 변수 + 지역 변수 이름이 같으면 헷갈릴 수 있음! 
- 변수 이름은 겹치지 않게!

## var 사용 변수 특징 ##

**JS `var` 변수 사용시 주의!!** 
- hoisting - 상황에 따라 변수의 선언 + 할당을 분리해서 선언 부분을 scope 가장 위로 hoist!
```js
var x = 100;

test();

function test() {
	document.write("x is " + x + ", y is " + y);
	var y = 200;
}
```
↳ 결과값은 `x is 100, y is undefined`
- y 값이 할당되지 않음!
- 이게 hoisting
```js
//컴퓨터는 이렇게 이해함

function test() {
  var y; // <-- this is hoisted to the top (only declared, no value yet)
  document.write("x is " + x + ", y is " + y);
  y = 200; // <-- assignment happens here
}
```
```js
// y = 200을 결과값에 나오게 하려면

var x = 100;

test();

function test() {
  var y = 200;  // ✅ move assignment BEFORE printing
  document.write("x is " + x + ", y is " + y);
}
// documnet.write을 declaration 이후에 사용하면 됨!
```

### Block안에서만 쓸 수 있는 변수 ###

## 함수의 return 더 알아보기 ##

	- 반환 위치 = 함수 호출한 위치 

```js

alert("두 수를 더한 값은 " + result + "입니다."); // result 변숫값= 화면에 표시

function addNumber(a, b) { // 함수 선언부로 넘어와 함수를 실행하는데, num1값은 a 변수로, num2 값은 b 변수로 넘겨짐
```

# 05-4 함수 표현시 #

## 익명 함수 ##
- 이름이 없는 함수 
	- 함수 자체가 식 (expression)
		- 익명 함수를 변수에 할당 + 다른 함수의 매개변수로 사용 가능!
```js
var add = function(a, b) { // 함수 선언 후 변수 add에 할당
    return a + b;
}

var sum = add(10, 20); // 익명 함수 실행 후 결괏값을 변수 sum에 저장
sum // 변수 sum값 확인
30
```
## 즉시 실행 함수 ##
- 함수를 정의함과 동시에 실행
	- 함수 선언 소스 전체를 괄호로 묶기 
	- 소스를 닫는 괄호 앞/뒤 인수가 들어갈 괄호 넣기
	- 즉.실.함은 식이다 = ; 끝에 필수!
```js
(
	function() {
	...
	}
)();

// OR

(
	function() {
	...
	}
());
```

and to shrink this, use:
```js
(function() {
	...
})();

//OR

(function() {
	...
}());
```

- 즉.실.함 = 변수에 할당 가능 & 함수에서 반환하는 값을 변수에 할당 가능
	- e.g. 함수에서 반환한 값을 result 변수에 할당한 후 콘솔 창에 표시
```js
var result = (function() {
    return 10 + 20;
}());
console.log(result);

30
```

- 매개변수 필요 함수 → function 다음 괄호 안에 매개변수 넣고 함수 정의 
	- 함수 끝에 있는 괄호에 실제 실행할 때 사용할 인수 넣고 실행 
```js
var result = (function(a, b) { // 매개변수 추가
	return a + b;
}(10, 20)); // 인수 추가
console.log(result);

30
```
## 화살표 함수 ##
=> 표기법 
- 익명 함수에서만 사용 가능!
### 매개변수가 없을 때 =>로 함수 선언 ##
- 매개변수가 없을 때, 매개변수 들어가는 괄호 비우기
```js
const hi = function() {
	return "안녕하세요?";
}

// ↑ this becomes ↓

const hi = () => { return "안녕하세요?" };
```

중괄호 안 함수 = 한 줄뿐 → 중괄호 생략
```js
const hi = () => "안녕하세요? ";

//e.g.
alert(hi());
```
### 매개변수 1개일때 ###
```js
let hi = function(user) {
	document.write(user + "님, 안녕하세요?");
}

// ↑ this becomes ↓ 

let hi = user => document.write(user + "님, 안녕하세요?");

// e.g. 
hi("경희");
```
↳ 매개변수의 괄호 생략 가능
### 매개변수 2개 이상일때 ###
```js
let sum = function(a, b) {
	return a + b;
}

 // ↑ this becomes ↓ 
 
let sum = (a, b) => { return a + b}

 // ↑ this becomes ↓ 
 
let sum = (a, b) => a + b;

//e.g.
document.write("두 수의 합 : " + sum(10, 20));
```
↳ 매개변수를 쉼표로 구분; 중괄호 안 식 = 하나, return is omitted

- 상세 설명 버튼 만들기
	- 이미지에 '상세 설명 보기'(내용 표시) 버튼 + '상세 설명 닫기' (내용 숨기기) 버튼 눌렀을 때 

```js
function showDetail() {
	document.querySelector('#desc').style.display = "block"; // 상세 설명 내용을 화면에 표시
	document.querySelector('#open').style.display = "none"; // 상세 설명 보기 버튼 감춤
}

function hideDetail() {
	document.querySelector('#desc').style.display = "none"; // 상세 설명 내용을 화면에서 감춤
	document.querySelector('#open').style.ldsiplay = "block"; // 상세 설명 보기 버튼 표시 
} 
```

## 여러 이벤트 처리기 연결 ##
- 쇼핑몰 사이트를 보면 작은 상품 이미지 위로 마우스 커서 가져갔을 때 큰 이미지가 표시되고, 작은 이미지를 누르면 상품 상세 페이지가 팝업 창으로 나타남
	- 구현 - onmouseover 이벤트 처리기 + onclick 이벤트 처리기 동시에 작동

1. click 이벤트 처리
	- 이미지를 눌렀을 때, alert 띄우기
```js
let coverImage = document.querySelector("cover");
coverImage.onclick = function() {
	alert("눌렀습니다");
}
```

2. mouseover, mouseout 이벤트 처리
	- 이미지에 마우스 커서를 올렸을 때, border 그려지기
```js
coverImage.onmouseover = function () {
	coverImage.style.border = "5px black solid";
}
```

+ 이미지에 커서를 때어냈을 때, border 없애기
```js
coverImage.onmouseout = function() {
	coverImage.style.border = "";
	}
```

↳ LOOK UP MATH FLOOR and MATH RANDOM FUNCTION!!!

- `Math.random()` - function
	- generates a random decimal number btw 0 and 1
	- NEVER returns 1

- `Math.random() * 100` - function
	- multiples that random decimal by 100
		- hence, the number btw 0 and 100

- `Math.floor(...)` - function
	- takes a decimal number & rounds it down to the nearest whole integer
		- e.g. 4.9 = 4; 4.1 = 4; 99.99 = 99

So `Math.floor(Math.random() * 100)` = integer btw 0 to 99

and `+1` = returns a random integer btw 1 & 100(inclusive)

```js
let count = 0;
let randomNumber = Math.floor(Math.random() * 100) + 1;
document.querySelector("#try").addEventListener("keydown", function (e) {
	if (e.key === "Enter") {
		e.preventDefault();
		finding();
	}
});

function finding() {
	let userNumber = document.querySelector("#try").value;
	if (userNumber >= 1 && userNumber <= 100) {
		if (randomNumber > userNumber) { // 컴퓨터 숫자가 클 경우 - 'UP' 표시
		document.querySelector("#display").innerHTML = "UP!";
	} else if (randomNumber < userNumber) { // 컴퓨터 숫자가 작을 경우 - 'DOWN' 표시
		document.querySelector("#display").innerHTML = "DOWN!";
	} else { // 컴퓨터 숫자를 맞혔을 경우
		document.querySelector("#display").innerHTML =
"<span style='color.:red'>맞혔습니다!</span>";
}

document.querySelector("#try").value = "";
count++;

document.querySelector("#counter").innerHTML = "시도 횟수: " + count + "회"; // #counter영역에 시도 횟수 (count) 표시
} else {
	alert("1과 100사이의 숫자를 입력하세요.");
	}
}
```

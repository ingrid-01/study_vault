## if & if...else ##
- 작성 순서대로 실행 X
- 명령 실행 순서 O
	- 조건에 맞는지만 확인한다면 if
	- 조건에 맞은 때와 맞지 않을 때를 구별해야 한다면 if...else
### if ###
- 소괄호 안의 조건이 true = JS source 실행
- 소괄호 안의 조건이 false = JS source 무시

if 가 true일때:
```js
if(true) {
    document.write("if문의 조건을 만족하여 이 문장이 실행되었습니다.");
}
```
↳ 문장이 브라우저 화면에 나타남

`if(false)` 일땐, 브라우저에 나타나지 않음.
### if...else ###
- 조건을 확인해서 조건에 맞을 때(조건식 결과값 true) 실행할 명령과 조건에 맞지 않을 때 (조건식 결괏값 false) 실행할 명령이 따로 있다면
	- if...else
- if 나 else 에서 실행할 명령 = 2+, {} 사용
	- {} 안에 실행 순서대로 명령 나열
```js
var number = prompt("숫자를 입력하세요.");
if(number < 0) {
    alert("0 이상의 수를 입력하세요");
}
else {
    document.write("입력한 숫자: " + number);
}
```
↳ 0 이상의 수 입력 = 실행
0 미만의 수 입력 = 실행 X

**if...else의 조건**
- if가 true면, else를 건너뜀
	- 즉, if를 무조건 먼저 확인한다

1. userNumber == null 조건 사용
	- `if(userNumber == null) {...}` // 취소 버튼을 눌렀을 때 실행할 명령
	- `else{...}` // 값을 입력했을 때 실행할 명령

2. userNumber != null 조건 사용
	- `if(userName != null) {...}` // 값을 입력했을 때 실행할 명령
	- `else {...}` // 취소 버튼을 눌렀을 때 실행할 명령

↳ 2 조건의 차이:
- 사용자가 숫자를 입력하는 횟수가 취소를 누르는 횟수보다 많을 것이다 
	- 즉, 2번 조건이 프로그램 실행 시간이 더 짧을 것
	- 즉, if...else를 만들 때 true가 될 경우가 많은 조건을 if에 넣어야한다! 
```js
<script>
	var userNumber = prompt("숫자를 입력하세요.");

```
## 조건 연산자 - ? 와 : ##
- 조건 하나 + true OR false 일때 실행 명령도 하나 
	- 조건 연사자 사용
	- ? 왼쪽에 조건
	- : 왼쪽에는 조건이 true일때, 오른쪽에는 조건이 false 일때 실행 명령

E.g. 60점 이상 = 통과; 아니면 실패 알림창
```js
var score = 75;
(score >= 60) ? alert("통과") : alert("실패");
```
↳ (score >= 60) → 조건
alert("통과")  → 조건이 true일때
alert("실패") → 조건이 false일때
## truthy 값과 falsy 값 ##
- 논리형 자료 값 = true & false; 딱 두 가지뿐. 
	- when a condition DIRECTLY evaluates to true = boolean true
		- `let name = true;` → name is true
	- when values aren't boolean true, but COERCED into true
		- e.g. non-empty strings ("hello", "0"), non-zero numbers (e.g. 1, -5, 3.14), objects (e.g. parentheses, curly braces), functions
		- `let username = "Alice";` → Alice is a truthy value = this conditions evaluates to true
	- All true values are truthy, not all truthy values are true
	- These cases are falsy, all others are considered truthy:
		- 0
		- "" - empty string
		- NaN
		- undefined
		- null

- truthy, falsy 값은 조건을 확인할 때 유용!
```js
var input = prompt("이름을 입력하세요.");
if(input) {
    alert("이름을 입력했습니다 : " + input);
}
else {
    alert("이름을 입력하지 않았습니다.");
}
```
↳ input에 값이 들어 있으면 truthy (true와 동일)
input에 값이 들어 있지 않으면 falsy (false와 동일)

# 04-3 명령 반복 실행 - for #
반복문 = 어떤 동작은 여러 번 실행
	- 반복문 중 제일 많이 사용되는 for
## 반복문 필요 경우 ##
e.g.
```js
var sum = 0;
sum += 1;
sum += 2;
sum += 3;
sum += 4;
sum += 5;
document.write("1부터 5까지 더하면 " + sum);
```
↳ this, but what if sum += 100?
이렇게 많은 양의 연산 처리에 반복문 필요

## for 작동 과정 이해 ##
- for는 값이 일정하게 커지면서 명령을 반복 실행할 때 편리!
```js
<script>
	var sum = 0;

	for(var i = 1; i < 6; i++) { // i = 1 부터, i = 5까지 총 5번 반복
		sum += i; // sum = sum + i; 를 줄여서 표현
	}
	document.write("1에서 5까지 더하면 " + sum);
</script>
```

**var i = ...; i < .... etc.**
- 카운터 변수 선언
	- for에서만 사용할 카운터 변수를 선언 

**i < 6**
- for 의 조건식
	- for안에 있는 소스를 실행할지 판단하는 조건
		- 조건 = true → 중괄호 안의 JS 소스를 실행
		- 조건 = false → for를 벗어나 다음에 나오는 소스 실행

**sum += i;**
- 반복 실행할 JS source
	- for에서 실행할 중괄호 안의 JS source

**i++**
- 카운터 변수 조절
	- 증감 연사자 (++)를 사용해 카운터 변수 조절
		- 위 소스에서는 중괄호를 실행한 이후 카운터 변수 i를 1만큼 증가 시킴

#### 카운터 변수 i ####

- for를 실행할 때 반복 횟수의 기준

# 04-5 특정 조건 따라 반복 - while, do...while #
- for 외에 반복문 - while, do...while
	- for : 횟수 정해진 반복
	- while, do...while: 특정 조건을 만족하는 동안만 명령 반복
## while과 do...while ##
**while** - 괄호 안 조건 만족 = 중괄호 안 명령 실행! 
```js
var i = 0
while (i < 10) {
    document.write('반복 조건이 true이면 반복합니다. <br>');
    i += 1;
}
```
↳ 윗 문장 → 10번 반복 

1. `i < 10` → 조건 검사
2. `document.write(...)` → 중괄호 안의 JS 소스 실행

**do...while**
- 조건이 while과 다르게, 뒤에 붙음!
- 일단 문장을 한 번 실행한 후 조건 확인
	- 즉, 결괏값이 false여도 일.단. 한 번은 실행
```js
var i = 0
do {
    document.write('반복 조건이 true이면 반복합니다. <br>'); // 명령 실행
    i += 1;
} while (i < 10); // 조건이 false가 되면 반복 종료
```

∴ 
	while = 조건부터 확인
	do...while = 일단 문장 실행한 후 조건 확인

어느걸 사용할 지는 프로그래머 취향대로...

### 어떤 반복문을 사용해야 할까? ###
- for → 초깃값이 있다 + 일정한 간격 반복
	- e.g. for(i = 0; i <10; i++) → 제일 편함
- while & do..while → 초깃값 or 반복 간격 X
	- 조건 맞지 않을 경우, 아예 명령 실행 X = while
	- 일단 명령 실행한 후, 상황에 따라 반복할 수도 있고 안 할 수도 있다 = do...while

# 04-6 반복을 건너뛰거나 멈추기 - break, continue #
- 흐름을 조절하고 싶을 때 
	- 예: 반복문의 특정 지점에서 반복 중단 OR 시작 지점으로 되돌아가고 싶다면?
	- break & continue 사용!
## break ##
- for로 별 문자 출력
```js
for (i = 0; i < 10; i++) {
	document.write("*");
	break;
}
```
↳ 10번 출력 안됨, 오류. 왜?

`break`에서 반복문 종료.
- break는 반복문의 흐름에서 바로 빠져나올 때 사용!
## continue ##
- continue → 주어진 조건에 맞는 값을 만났을 때, 실행하던 반복 문장을 건너뛰고 반복문의 맨 앞으로 되돌아갑니다. 
	- 즉, continue 다음 명령 → 건너뜀
```js
for (i = 0; i < 10; i++) {
    document.write("*");
    continue;
    document.write("continue문 때문에 이 문장을 건너뜁니다.");
}
```
## 짝수 더하기 프로그램 ##
- continue + if 활용해 짝수만 골라서 더하는 프로그램
	- first, if로 더할 수 = 짝수 or 홀수 확인
	- 숫자가 홀수 = continue로 반복 흐름 끊고 처음으로 되돌아가기
	- 짝수 = 숫자를 더하기 
	- 반복문으로 원하는 숫자까지 짝수만 더한 값을 얻을 수 있음!

```js
<script>
	var n = 10;
	var sum = 0;
	
	for (var i = 1; i <= n; i++) { // i <= 10까지 반복
		if (i % 2 == 1) { // i가 홀수일 경우 실행
			continue;
		}
	// i가 짝수일 경우 실행
	}
	sum += i;
	
	document.write(i + " ------ " + sum + "<br>");
</script>
```

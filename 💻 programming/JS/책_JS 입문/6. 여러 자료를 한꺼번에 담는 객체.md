# 06-1 객체 #

- 복합 자료형 = 객체
	- 겍체 안에 숫자, 문자열 등 여러 가지 자료형이 포함됨
## 객체 ##
- 하나의 변수에 다양한 정보 담는 자료형 = 객체
	- 예: 제목 - 자바스크립트, 저자:홍길동, 쪽수: 500pg 등 = book 객체
```js
let book = {
	title: "JavaScript",
	author: "John Smith",
	pages: 500,
	price: 15000
}
```

- 회원이란 변수 = 객체
	- 이름, 아이디, 비밀번호, 기입 날짜, 생년월일 등
## JS 객체 종류 ##
- Object = JS에서 인식할 수 있는 모.든. 대상
### 내장 객체 ###
- Built-in Object
	- 미리 객체로 정의된 것
	- 예 -
		- Date 객체 → 현재 시각 사용 가능
		- Number, Boolean, Array, Math 등
### 문서 객체 모델 (DOM) ###
- Document Object model
	- 객체를 사용하여 웹 문서 관리하는 방법
		- 웹 문서 + 포함 된 이미지, 링크, 텍스트 필드 등 각각 별도 객체
			- Document 객체
			- Image 객체
### 브라우저 객체 모델 ###
- 브라우저의 - 창 크기, 주소 표시줄 등
	- Navigator 객체 - 사용중인 브라우저 종류 + 버전
	- History 객체 - 브라우저 방문 기록
	- Location 객체 - 주소 표시줄 정보
	- Screen 객체 - 화면 크기 정보
- 알림 창 표시 + 사용자 동작 따라 사이트 이동 가능

### 사용자 정의 객체 ###
- 사용자가 필요할 때 자신의 객체를 정의할 수 있다
	- 예: 위에 book 변수 = 사용자 정의 객체
## 객체의 속성 + 매서드 ##
- 객체 값 담고 있는 곤간 = 속성 (Property)
	- 속성은 내장 객체에도 있음
	- 겍체 속성 값 가져올 때 → 마침표 (.) 찍고 + 속성 이름 적기
```js
navigator.vendor // 속성 + 함수
"Google Inc." // 저장된 브라우저 제조업체

navigator.onLine // 인터넷 연결 유무
true / false // 속성 값 (true면 연결됨, false면 연결되지 않음)
```

- 객체의 동작 선언하는 함수 = 메서드 (Method)
	- `window` → 콘솔에 입력 = window 객체의 메서드
	- `window.alert` → alert 창 = Window객체의 메서드
↳ Window객체의 함수 = 이름만 사용 가능
	- 예: alert() - instead of window.alert()
	- open() → 새 탭 창 열기 메서드
## 객체의 프로토타입 + 인스턴스 ##
- 이미지를 JS에서 다룰 때, Image() 객체 사용
	- Image() 객체 → 모든 웹 이미지가 공통으로 가진 속성과 기능을 모아놓은 것
	- 기본 틀 = Prototype

- Object:
	- individual instance of data structure (own properties & values)
- Prototype:
	- blueprint/template object → instance can inherit from
- Instance:
	- Prototype으로 찍어낸 객체 
	- "개별 객체"
## 객체의 인스턴스 만들기 ##
- 객체를 똑같이 찍어 새 객체를 만들 때:
	- new 예약어 사용
	- new + prototype 객체 이름 + ()
- 예:
	- 날짜 + 시간 관련 프로그램 작성하려면-
		- Date객체의 instance 만들기 + instance 변수 저장 = 프로그램에 사용
```js
let now = new Date()

Wed Nov 05 2025 17:11:55 GMT+0900 (Korean Standard Time)
```
↳ new 예약어로 Date객체의 instance 만들고
now 변수에 저장 → now 변수값 = 현재 날짜와 시간
- now → 이제 Date 객체의 instance

```js
now.toLocaleString()

'05/11/2025, 17:11:55'
```
↳ toLocaleString() = 현재 날짜와 시간 정보를 로컬 형식으로

## 내장 객체로 Random Number (무작위 수) 프로그램 만들기 ##
- 내장 객체 중 하나: Math
	- 삼각함수, 로그 함수 등 수학 연산 함수 가지고 있는 내장 객체
- 자주 사용하는 Math 함수:

| 함수       | 설명                                                       |
| -------- | -------------------------------------------------------- |
| abs(x)   | absolute - 숫자의 절댓값을 반환한다                                 |
| cbrt(x)  | cube root - 숫자의 세제곱근을 반환한다                               |
| ceil(x)  | ceiling - 인수보다 크거나 같은 수 중, 가장 작은 정수를 반환 (숫자의 소수점 이하를 올림) |
| floor(x) | floor - 인수보다 작거나 같은 수 중, 가장 큰 정수를 반환 (숫자의 소수점 이하를 버림)    |
| random() | 0과 1사이의 무작위 수를 반환                                        |
| round(x) | 숫자에서 가장 가까운 정수 반환 (숫자의 소수점 이하를 반올림함)                     |
- Math.random() → 무작위 수 반환
	- 1-100 사이 수 구하려면:
```js
Math.random() * 100 + 1 // +1 changes range to 1 to 101; so we get a number btw 1 to 100
69.77507422327 
```
- 그리고 정수 만들기:
```js
Math.floor(Math.random() * 100) + 1
```

⚠️ Math.random 처럼 대문자로 시작하는 객체 = 인스턴스 사용 X
- 바로 속성 and/or 함수 사용
# 06-2 사용자 정의 객체 만들기 #
JS에서 객체를 만드는 방법 중 가장 많이 사용하는 객체 리터럴 + 생성자 함수
## 리터럴 표기법을 사용해 객체 만들기 ##
- Literal → 자료 표기 방식
	- 변수 선언 동시에 값을 지정
	- 예: `let a = 10;`
- 객체 리터럴 표기법
	- 객체 선언 동시 값을 지정
		- 중괄호 {} 안에  '속성 이름: 값' - 하나의 쌍으로 지정
		- 속성 여러 개 = '속성 이르: 값' + ',' 로 구분
	- 함수 정의
		- '함수 이름: function() {...}'
	- 객체를 정의해 변수에 할당할 때, 닫는 중괄호에 ; 붙이기!!!

```js
let book = { // book - 객체
	title: "JavaScript",
	author: "Go Teacher", // author - 속성 이름, "Go.." - 속성 값
	pages: 500,
	price: 15000,
	info: function() { // info - 함수 이름; 나머지 - 함수
		alert(this.title + "책의 분량은 " + this.pages + "쪽입니다.");
	}
};
```

`this` → object that is currently executing the code
- `this` depends on how a function is called
	- doesn't have fixed value
	- so ↑ here, `this` = `book`
```js
book.title
"JavaScript"

book.info()
// alert pops up saying
JavaScript책의 분량은 500쪽입니다. 
```

- 프로그램에서 book 객체를 사용하다가 새로운 속성 추가하기:
	- `field` 속성 추가 + 값 넣기
```js
book.field = "IT"
"IT"

book
	...
	field: "IT"
```
### 연습: 리터럴 표기법으로 장난감 정보 객체 만들기 ###
- 객체 만들 때, 공통 속성 + 함수 적다면 => 여러 객체 묶을 필요 X
	- 각 객체 따로 정의할 때, Literal 표기법 사용

```js
let toyRobot = {
	productId: "123-12",

toyRobot.showStock();

//Robot 제품은 10개 남아있습니다. 
```
## 생성자 함수를 사용해 객체를 만드는 법 ##
- 리터럴 표기법:
	- 정해진 값을 가진 객체를 한 번만 만듦
	- 이걸 이용하여 도서 관리 프로그램을 만드려면:
		- 매번 객체와 똑같은 속성 + 함수 입력 & 책마다 그 값을 다르게 넣어 새롭게 객체를 만들어야함 
			- !!!개빡침!!!

BUT → 항상 필요한 속성 + 함수 틀로 만들면?
	필요할 때마다 틀 복제한 instance 사용가능
- "생성자 함수" 사용하여 객체 만들기

- 생성자 함수 (constructor function) → 함수이기에 function 예약어 사용해 선언

예시 - Book 객체 만들기:
```js
function Book(author, pages, price, title) {
	this.author = author;
	this.pages = pages;
	this.price = price;
	this.title = title;
}
```

instance 만들기:
```js
jsBook = new Book("홍길동", 500, 15000, "자바스크립트")
// > Book {author:.., pages:..., price:..., title:...}

jsBook.title
// "자바스크립트"
```
### 연습: 생성자 함수로 도서 목록 객체 만들기 ###

```js
function Book(title, author, volume, price) {
	this.title = title;

	this.volume = volume;
```

```js
let html = new Book('웹 표준의 정석', 'Ko', '608', '28,000');
```

↳ 이렇게 여러 객체가 저장되었을때, 책 제목만 골라 배열로 작성하기:

```js
let bookList = [html, youtube, python];
```

- 책 제목만 골라서 표시하기

```js
document.write("<h1>책 제목으로 살펴보기</h1>");
```

# 06-3 Date 객체를 활용해 기념일 계산기 만들기 #

## Date 객체 ##
- 내장 객체 = Date
	- 현재 날짜 + 시간 알 수 있음
	- 특정 날짜 _ 시간까지 얼마나 남았는지 등 알 수 있음

- Date 객체 사용하려면 instance 만들어야함!
### 현재 날짜 정보를 가지는 Date 객체 만들기 ###
- Date instance → `new` 예약어 + `Date` 객체
	- 현재 날짜 + 시간이 담김
```js
new Date()
// browser shows ↓ 
Sun Feb 25 2018 13:55:19 GMT+0900 (한국 표준시)
```
### 특정 날짜나 시간 정보를 가지는 Date 객체 만들기 ###
- Date + () 안에 날짜 정보 입력
	- ("year, monthIndex, day") OR
	- ("year-month-day")
```js
new Date("2025-11-06")

//OR

new Date(2025, 10, 06)

// shows same results
```

- 연도 + 월까지만 = 해당 월 1일, 오전 9시 설정
```js
new Date("2025-02")
//
Sat Feb 01 2025 09:00:00 GMT+0900 (Korean Standard Time)
```

- 시간 정보 지정 = 날짜 다음 T추가
```js
new Date("2018-02-25T18:00:00")
//
Sun Feb 25 2018 18:00:00 GMT+0900 (Korean Standard Time)
```

- UTC (국제표준시) 지정 = 끝에 Z
```js
new Date("2025-11-08T18:00:00Z")
//
Sun Nov 09 2025 03:00:00 GMT+0900 (Korean Standard Time)
```
↳ 객체에 담기는 실제 시간 = 한국 표준시
### JS 날짜/시간 입력 방식 ###
1. ISO 형식 - YYYY-MM-DDTHH:MM:SS
```js
new Date("2025-11-08T18:00:00")
```
2. 짧은 날짜 - MM/DD/YYYY
```js
new Date("08/11/2025")
//
Mon Aug 11 2025 00:00:00 GMT+0900 (Korean Standard Time)
```
3. 긴 날짜 - MMM/DD/YYYY
```js
new Date("Nov 08 2025")
Sat Nov 08 2025 00:00:00 GMT+0900 (Korean Standard Time)
```
## Date 객체의 주요 함수 ##
- Date 객체 만든 후 → Date 객체 안에 정의되어 있는 함수 사용가능
		- 함수 앞에 get / set 붙음
		- get: 가져오다
		- set: 두다, 설정하다

- getSeconds() - 초 정보를 가져오고
- setSeconds() - 초 정보를 원하느 값으로 설정한다

| 날짜/시간 정보 가져오는     | 설명                                | 날짜/시간 정보 설정       | 설명                       |
| ----------------- | --------------------------------- | ----------------- | ------------------------ |
| getFullYear()     | 날짜 정보에서 - 연도 정보 4자리 수로 표시         | setFullYear()     | 연도 4자리 수로 설정             |
| getMonth()        | 월 정보 0~11수로 표시 (0 = 1월, 11 = 12월) | setMonth()        | 0~11 월 표시                |
| getDate()         | 일 정보 가져오기                         | setDate()         | 1~31 일 설정                |
| getDay()          | 요일 정보 가져오기 (0 = 일, 6 = 토)         |                   |                          |
| getTime()         | 1970.1.1 자정 이후 → 밀리초 표시. 1/1000초  | setTime()         | 1970.1.1 자정 이후 → 밀리초 표시. |
| getHours()        | 0~23 시 표시                         | setHours()        | 0~23 시 설정                |
| getMinutes()      | 0~59 분 표시                         | setMinutes()      | 0~59분  설정                |
| getSeconds()      | 0~59 초 표시                         | setSeconds()      | 0~59초  설정                |
| getMilliseconds() | 0~999                             | setMilliseconds() | 0~999                    |
## 현재 날짜 + 시간 정보 가져오기 ##

- Date 객체 인스턴스 만들고 + now변수에 저장
```js
let now = new Date()
now
//
Sat Nov 08 2025 17:18:07 GMT+0900 (Korean Standard Time)
```

- now 객체에서 날짜 정보만 가져오면 getDate() 사용
```js
now.getDate()
//
8
```

- now 객체에서 요일 정보
```js
now.getDay()
//
6 // Saturday
```

- getTime → 시간정보 표시
```js
now.getTime()
//
1762589887508 // 
```
↳ 1970.1.1 00:00 기준으로 now의 시간까지 얼마나 흘렀는지 milliseconds로 나타낸것

⚠️ Unix Epoch = 1st of Jan, 1970 00:00:00 UTC (Coordinated Universal Time)
- FIXED starting point in time that computers use to measure & represent ALL OTHER DATES & TIMES
- Computers think in NUMBERS, not "years" or "months" = time stored in number of SECONDS
- It's a "ZERO POINT"

EPOCH : a significant starting point in history / new era
- zero point from which ALL times values are measured

UNIX: 1960s/70s - Unix OS was being built
- Unix measure all times as = seconds since 1st of Jan, 1970, UTC

- 오늘부터 50일 - 계산방법
```js
now.setDate(now.getDate() + 50) // 오늘 날짜를 가져와 50을 더한 후 다시 날짜로 입력
//
1766916419398 // 결과값 = 밀리초로 표시

now // now객체 확인
//
Sun Dec 28 2025 17:18:07 GMT+0900 (Korean Standard Time) // 50일 후의 날짱 + 시간 정보
```

- now객체에 있는 월 정보 콘솔 창에 표시
```js
console.log(now.getMonth() + 1)
//
12
```

## 기념일 계산기 만들기 ##

### 며칠 만났는지 알아보기 ###
- 두 날짜 픽 & 며칠 흘렀는지 계산

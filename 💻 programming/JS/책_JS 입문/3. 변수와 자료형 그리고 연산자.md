# 03-1 변수란 무엇일까? #
JS의 변수 (variable) = 변하는 값을 저장한다 → 상황에 따라 다른 값을 입력받아야할때
## 미리보기 - 나이 계산하기 ##
- 올해 연도와 태어난 연도 입력 → 사용자의 나이를 계산해 주는 프로그램
### 수식으로 알아보는 상수와 변수 ###
- 나이를 계산하는 수식 
	- 나이 = 올해 연도 - 태어난 연도 + 1
	- 나이, 올해 연도, 태어난 연도 → 언제든 값이 변하므로 '변수' (variable)
	- +1 → 항상 값이 일정하므로 '상수' (const)
		- 한국식 나이 계산법
## 변수를 선언하는 규칙 세 가지 ##
- 변수 = cpu가 구별할 수 있게 이름 필요
	- '변수를 선언한다'
	- `var apple;`
	- 이를 행하기 위해 규칙 필요

1. 이름을 의미 있게
	- a, b 같은 변수는 나중에 왜 만들었는지 알기 어렵다 
	- currenYear, birthYear
`var age, var currentYear, var birthYear`

2. 여러 단어를 연결한 변수 이름은 CamelCase
	- 첫 번째 단어 - lower case
	- 두 번째 단어 - upper case
	- can use _ btw the words: current_year, birth_year

3. 선언할 수 없는 이름도 있다
	- 변수 이름의 첫 글자 = 문자, _ or $
	- 그 후 = 문자, _ , $ or 숫자
	- 이 패턴을 벗어나면 안됨

## 변수에 값이나 식 저장하기 ##
- 변수에 값 저장 → 변수 오른쪽에 = 기호를 붙이고 오른쪽에 저장하고 싶은 값이나 식을 쓴다 
	- 변수 선언하면서 저장
	- 변수 선언, 그 다음에 저장
```js
var apple = "yummy"; // 변수를 선언하면서 값을 저장
var banana;
banana = "long and yummy"; // 변수 선언, 다음에 값을 저장
```

- 변수에 식을 저장할 경우: 변수 미리 선언
```js
var apple = 2000;
var banana = 10000;
var total = apple + banana;
```
## 나이 계산 프로그램 만들기 ##

1. 변수 선언하기
```js
<script>
	functon calc() {
		var currentYear = 2025;
```

- `document.querySelector("#result").innerHTML`
	- document에서 선택자 (querySelector) 를 사용하여 id 값이 result인 태그("#result")를 선택하고 HTML에 삽입한다(innerHTML)

2. 사용자 입력값 변수에 할당하기
```js
<script>
	function calc() {

		var birthYear = prompt("태어난 연도를 입력하세요.", "YYYY");
		var age;
		age = currentYear - birthYear + 1;
		document.querySelector("#result").innerHTML = "당신의 나이는" + age + "세입니다.";
	}

```
- prompt로 사용자가 값을 입력

`var`말고 `let`과 `const` 예약어 사용
- `let`
	- 변수는 블록을 (curly brace안에) 벗어나면 사용할 수 없음
- `const`
	- 상숫값을 선언할 때 사용함

```js
function calc(){
	const currentYear = 2019;

}
```
# 03-2 자료형 이해하기 #
- 변수를 저장하는 값의 형태를 의미하는 자료형 (Data Type)
## 자료형의 종류 ##
- 자료형 = 컴퓨터가 처리하는 자료의 형태
	- aka 데이터 타입, 데이터형, 데이터 유형

1. 기본형 (primitive)
	- number (숫자형)
	- string (문자형)
	- boolean (논리형)
	- undefined - 자료형 지정 X; e.g. 변수 선언 BUT 값 정의 X
	- null - 값 유효 X
2. 복합형
	- array (배열) - 하나의 변수에 여러 값을 저장
	- object (객체) - 함수와 속성이 함께 포함

### 자료형을 확인하는 방법 - typeof 연산자 ###
- JS는 변수에 저장하는 값에 따라 자료형이 결정 됨
	- e.g. 
		- value var → 10 → data type = number
		- "10" → data type = string
		- 그래서, 변수/값의 자료형을 찾기 위해 → typeof 연산자(operator) 사용
```js
typeof 1990;
"number"
typeof "hello";
"string"
```
## 숫자형 - 정수 (integer) ##
```js
var birthYear = 2000;
undefined
typeof birthYear;
"number"
```

- 소수점 없는 숫자
- 10진수, 8진수, 16진수 세 가지 유형으로 나뉘기도 함

**8진수*** 
- 0~7로 표현하는 수
- 10진수와 구별하기 위하여 숫자 0을 맨앞에 붙임.
- 012, 013, 01000 (10진수로 10, 11, 512)

**16진수**
- 숫자 0~9, 알파벳 A~F (대문자, 소문자 둘 다)
- 프로그래밍 할 떄 가장 많이 사용
- 10진수와 구별하기 위해 0x (0X)를 맨 앞에 붙임
- 0xfff, 0xFFF, 0Xfff, 0XFFF (넷 모두 10진수로 16^3-1=4095)
## 숫자형 - 실수 (real number; decimals) ##
- 소수점이 있는 숫자
```js
var average = 17.5;
undefined
typeof average;
"number"
```

BUT
- JS에서 실수 계산
	- 0.1 + 0.2 = 0.30000000000000004
	- because JS 0.1, 0.2 = 2진수로 변환해서 계산
	- **정밀한 숫자 계산 프로그램 만들 때 주의**
## 문자형 ##
- single or double quotation mark로 묶은 자료
	- 숫자도 따옴표로 묶으면 = string
```js
var message = "두잇공부중";
undefined
typeof message;
"string"
```

- 따옴표 중복 방지
```js
document.write("<span class='num'>", 2018, "</span>");
undefined
```
- num is used with single quotes because the whole var is in double-quotations
## 논리형 (Boolean) ##
- True & False 값을 표현
	- 조건 확인 시 많이 사용
	- e.g. age 변수 → 20보다 크면 성인, 20보다 작으면 미성년자 => age > 20 조건을 입력 = 결과값이 true or false

```js
var age = 21;
undefined
age > 20
true
age < 20 
false

typeof true;
"boolean"
typeof false;
"boolean"
```
## undefined와 null ##
- undefined - 자료형이 정의되지 않았을 때 = 처음부터 변수에 값이 할당되지 않았다 =/= 자료형을 가진다
- null - 처음에 할당된 값이 더는 유효하지 않음
```js
var first;
undefined
var second = 10;
undefined
second = null;
null
```
↳ var first = 값이 할당되지 X = undefined
var second = 값이 10에서 null로 변함 = 10이 유효 X
- cpu는 처음부터 없던 경우 vs. 더 이상 유효하지 않은 경우를 나눈다
- 보통 변수를 초기화할 떄 null을 많이 사용함

## 배열 ##
- 하나의 변수에 여러 값을 저장할 수 있는 배열 유형
	- e.g. 여러 칸으로 나뉜 알약 통 = 요일별 나눔

e.g. 계절 이름을 변수에 할당하는 프로그램

```js
var seasons = ["봄", "여름", "가을", "겨울"];
seasons
["봄", "여름", "가을", "겨울"]
> (4)
```
↳ 저장할 자료를 쉼표로 구분, 대괄호로 묶음 = 배열 선언

빈 배열도 배열 선언
```js

```
### array 특정 값 ###
- 배열 이름 + 대괄호 + 0부터 숫자 = 해당 위치의 값만 가져옴
- 여기의 사용한 0, 1, 2, 3 = 배열 인덱스
	- 인덱스는 항상 0으로 시작

# 03-3 연산자 이해하기 #
- 할인 가격 계산해 주는 프로그램
## 미리보기 - 할인 가격 계산하기 ##
- 원래 가격과 할인율을 입력하면 얼마를 절약하는지, 얼마에 물건을 살 수 있는지 출력
## 기초 산술 연산자 ##
1. 사칙 연산자
	- 더하기 (+)
	- 빼기 (-)
	- 곱하기 ( * )
	- 나누기  ( / )
2. 나머지 연산자
	- 나머지 (%) - remainder
3. 증감 연산자
	- 증가 (++) - 변숫값을 1만큼 증가
	- 감소 (--) - 변숫값을 1만큼 감소

### 연산자 (Operator) & 피연산자 (Operand) 구별 ###
- A + B → A, B are operand, + is an operator
### 사칙 연산자 이해 ###
```js
10 + 20;
30
10 - 20;
-10
10 * 20;
200
10 / 20;
0.5
```
### 나머지 연산자 이해 ###
```js
15 % 2;
1
```
### 증감 연산자 이해 ###
```js
let a = 10;
let b = a++ + 5;
let c = b--;
a
11
b
14
c
15
```
```js
let a = 10;
let b = a++ + 5;
b
15
let c = 10;
let d = ++c + 5;
d
16
```

- `b = 10 + 5` → `b` becomes 15
- THEN `a` becomes 11
	- post-increment a → b is calculated first before the increment

- `d = 11 + 5` → `d` becomes 16
- pre-increment c → c is incremented THEN d is calculated
## 할인 가격 계산 프로그램 만들기 ##
1. **연산자 활용**
```js
<script>
	function showPrice() {
		let originPrice = 10000;
		let rate = 25;
		document.querySelector("#showResult").innerHTML = "상품의 원래 가격은 " + originPrice + "원이고, 할인율은 " + rate + "%입니다.";
	}
</script>
```
↳ 출력값 = '상품의 원래 가격은 10000원이고, 할인율은 25%입니다.'

- 초기 모델 완성!

2. **입력 상자에 입력한 값 받기**
```js
<script>
	function showPrice() {
		let originPrice = document.querySelector("#oPrice").value;
		let rate = document.querySelector("#rate").value;
		document.querySelector("#showResult").innerHTML = "상품의 원래 가격은 " + originPrice + "원이고, 할인율은 " + rate + "%입니다.";
	}
</script>
```
↳ 입력한 값 + 할인율 출력됨

```js
<script>
	function showPrice() {
		let originPrice = document.querySelector("#oPrice").value;
		let rate = document.querySelector("#rate").value;
		let savedPrice = originPrice * (rate / 100);
		let resultPrice = originPrice - savedPrice;
		document.querySelector("#showResult").innerHTML =
		"상품의 원래 가격은 " +
		originPrice +
		"원이고, 할인율은 " +
		rate +
		"%입니다. " +
		savedPrice +
		"원을 절약한 " +
		resultPrice +
		"원에 살 수 있습니다.";
	}
</script>
```
↳ 입력한 값 + 할인율 + 계산때려 출력됨. 완성!
# 03-4 연산자 깊게 알아보기 #

## 할당 연산자 응용 ##
- Assignment Operator (할당 연산자, =)는 JS에서 가장 많이 사용하는 연산자
	- 산술 연사자( +, -, * , / , %) 랑 묶어 사용하기
```js
var x = 10,
x += 10;
var y = 20;
y += x;
// 풀이//
```
```js
var x = 10;
x = 10 + x;
20
var y = 20
y = 20 + y
40
```
↳ ∴ x, y (변수)를 사용하여 산술 연산자 (+) 를 한 번 더 사용

| 할당 연산자 응용 | 예      | 의미        |
| --------- | ------ | --------- |
| +=        | y += x | y = y + x |
| -=        | y -= x | y = y - x |
| *=        | y *= x | y = y * x |
| /=        | y /= x | y = y / x |
| %=        | y %= x | y = y % x |

	- backtick으로 묶고 + 값은 ${} 사이 표시

## 문자형과 숫자형의 연산 이해 ##
### 더하기 빼기 ###
- 숫자형 자료 + 문자형 자료 = 
```js
var numVar = 100;
var strVar = "50";
numVar + strVar
"10050"
```
↳ 두 숫자를 연결한 문자열 => +기호 = 연결 연산자 

```js
numVar - strVar
50
```
↳ - 기호 = 산술 연사자; changes any string to number if possible

### 곱하기, 나누기, 나머지 ###
- 나머지 연산 = 문자형 자료를 모두 숫자로 자동 인식
	- BUT 파이썬 → 곱하기 연산자 = 문자열 복제 + 연결
	- 즉 언어마다 다르게 처리
## 비교 연산자 ##
- Comparison Operators = 2 값을 비교 (T or F)
### == 와 != 연산자 ###
== 두 수 (변수)가 같은지 확인해준다
!= 그 반대의 역할 (다른자) 확인해준다
```js
var a = 3, b = 4;
a == b
false
b != 6
true
```
### <와 <=, >와 >= 연산자 ###
- <와 <= 연산자 → 왼쪽보다 오른쪽이 더 작은지 또는 작거나 같은지 확인
- >와 >= 연산자 → 왼쪽보다 오른쪽이 더 큰지 또는 크거나 같은지 확인
```js
a < 3
false
a <= b
true
a > b
false
b >= 2
true
```
### == 연산자와 === 연산자 ###
- == , === → 둘 다 왼쪽 값과 오른쪽 값이 같은지 비교
```js
10 == "10"
true
10 === "10"
false
```
- == → 문자형 자료, 숫자형 자료 = 자동 변환 후 비교
- === → 두 자료 변환 허용 X

⚠️ 보통 === 더 많이 사용! == 는 자동 변환되어 false 처리 할 것을 true로 처리할 수 있음

## 문자끼리 비교 - 아스키 (ASCII) 값 ##
- compare btw "a" and "b" OR "A" and "a"
	- "A" = 65, "a" = 97
	- "A" > "a" → false
- when needing to compare 
	- theasciicode.com.ar = ASCII code table available

## 논리 연산자 ##
- true, false를 처리하는 연산자 = true, false가 피연산자로 됨
- aka. Bool or Boolean 연산자
### OR ###
- || → 왼쪽과 오른쪽 중 하나가 참이면 true
- 하나라도 맞아라!
```js
var a = 10, b = 20;
a > 10 || b > 20
false
a <= 10 || b > 20
true
a < 10 || b <= 20
true
a <= 10 || b <= 20
true
```
### AND ###
- && → 왼쪽과 오른쪽 모두 참인 경우만 true
- 하나라도 틀리면 false!
```js
var a = 10, b = 20;
a > 10 && b > 20
false
a <= 10 && b > 20
false
a < 10 && b <= 20
false
a <= 10 && b <= 20
true
```
### NOT ###
 - ! → true or false를 반대로 뒤집
```JS
var isAdult = false;
isAdult
false
isAdult = !isAdult;
isAdult
true
```
## 서로 다른 연산자를 계산하는 순서 ##
- 다른 분류일 떄 계산하는 순서
	- 단항 → 산술 → 비교 → 논리 → 할당

- 같은 분류일 때 계산하는 순서

|        | 1st | 2nd  | 3rd | 4th | 5th | 6th | 7th |
| ------ | --- | ---- | --- | --- | --- | --- | --- |
| 단항 연산자 | !   | ++   | --  |     |     |     |     |
| 산술 연산자 | *   | /    | %   | +   | -   |     |     |
| 비교 연산자 | <   | <=   | >   | >=  | ==  | !=  | === |
| 논리 연산자 | &&  | \|\| |     |     |     |     |     |
| 할당 연산자 | =   | +=   | -=  | *=  | /=  | %=  |     |

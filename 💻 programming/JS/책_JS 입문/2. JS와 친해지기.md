# 02-4 나의 첫 번째 JS Program #
## '미리보기' 인사하는 브라우저 ##
```js
<script>
	var name = prompt("이름을 입력하세요.");
	document.write("<b><big>" + name + "</big></b>님, 환영합니다.");
</script>
```

- WB에는 HTML분석기 (HTML Parser), CSS 분석기 (CSS Parser), JS 해석기 (Javascript Interpreter) 포함
	- 각각의 분석기, 해석기 → 첫 번째 줄부터 한 줄씩 내려가며 분석하고 해석한다
		- HTML 분석기 reads = `<!DOCTYPE html>`
		- CSS 분석기 reads = `<style></style>`
		- JS 해석기 reads = `<script></script>`
# 02-5 JS 입력과 출력 #
## Chrome Browser Console Tool과 함께 공부하기 ##
- 콘솔 툴 열기 
	- command + option + J
	- 결괏값이 없을 경유 - `undefined`라고 표기

### 사용자 입력값 받기 - prompt () 함수 ###
- 사용자에게 값을 입력받을 때 = 가장 쉽게 사용 가능 
- `prompt ()`
	- 사용자가 값을 입력할 수 있도록 작은 창을 만듦
	
```js
> prompt();
< "hello"
```
↳ prompt 창에 입력 값 = 콘솔 창에 입력값 확인
- 주로 어떤 내용을 입력해야 하는지 안내할 때 많이 사용

e.g.
`prompt("이름을 입력하세요.");`
↳ 프롬프트창에 입력값이 보임

`prompt("이름을 입력하세요.", "홍길동");`
↳ 1. 프롬프트창에 입럭값; 2. 텍스트 필드 안에 기본 값 표시

### 알림 창으로 출력하기 - alert() 함수 ###
- WB 화면에 간단한 알림 내용 포시 = `alert()`
	- `alert("환영합니다");`

### WB 화면에 출력하기 - document.write() 함수 ###
- 괄호 안에 내용을 chrome browser화면에 표시 
	- 즉, 결과값을 브라우저의 출력하는 용도
	- e.g. 
	 `var name = prompt("이름: ");`
	 `document.write(name + "님, 어서오세요!");`

	↳ var name = prompt 는 팝업창 + document.write은 브라우저에 보여짐

- `document.write()` 는 `document.`가 앞에 붙음
	- document 객체에 포함 → what we are writing is included in the document

### 콘솔에 출력하기 - console.log() 함수 ###
- 괄호 안의 내용 = 콘솔 창에 출력 
	- document.write 함수랑 같은 사용 방법
	- e.g.
		`var name = prompt("이름: ");`
		`console.log(name + "님, 어서오세요!");`

		`백두산님, 어서오세요!`

# 02-6 JS source를 작성할 때 지켜야 할 규칙 #
- 꼭 지켜야 할 6가지 규칙!!!

### 규칙 1 - 대소문자를 구별하여 소스를 작성한다 ###

- JS는 대손문자 구분 = sum, Sum, SUM = ALL DIFFERENT
### 규칙 2 - 읽기 쉽게 들여 쓰는 습관을 들인다 ###

- using Tab
### 규칙 3 - 세미콜론으로 문장을 구분하다 ###

`var num1`
`var num2`

`var num1;`
`var num2;`

↳ 두 문장의 실행 결과 = 같음 BUT 실수로 줄바꿈 하지 않으면 오류 발생
	e.g.  `var num1 var num2`
### 규칙 4 - JS 소스에 메모를 하려면 comment ###

한 줄 주석 - `//`
```js
var today = new Date(); // 날짜를 가져온다
var h = today.getHours(); // 시를 추출한다
```

여러 줄 주석 - `/* */`
```js
/*
	현재 날짜를 가져와
	시와 분, 초로 추출하고
	화면에 표시하는 스크립트
*/
function startTime() {...}
```
### 규칙 5 - 식별자(identifier)는 정해진 규칙을 지켜 작성한다 ###

- 식별자 (identifier) = 변수 (variable), 함수 (function), 속성 (attribute) 구별법
`var name = prompt("이름을 입력하세요.")` → name = identifier

- 식별자의 첫 글자 = 영문자, _ or $
- 그 이후 = 영문자, _ , $ or 숫자
- 공백 X
- 단어 사이사이: - or _ or 붙여서 사용할땐 첫 단어 (소문자) 두번째 단어 (대문자)

```js
num1 // 영문자로 시작하는 식별자
_doSomething // 밑줄 (_)로 시작하는 식별자
checkTime() // 두 단어로 만든 식별자 
```
### 규칙 6 - 예약어(keyword)는 식별자로 사용할 수 없다 ###

- 예약어 (keyword) = JS에 먼저 등룍된 요소

# 필수 개념 & 문법 퀴즈 #

1. `<script>` tag는 HTML 문서의 어디에나 넣을 수 있습니다. O
2. `<script>` 태그는 한 문서 안에서 여러 번 사용할 수 있습니다. O
3. JS 소스에서는 대소문자를 구별하지 않습니다. X
4. 예약어는 식별자로 사용할 수 없습니다. O
5. HTML 문서에 연결하는 외부 JS 파일의 확장자는 ==.js== 입니다. 
6. 간단한 JS는 크롬 브라우저의 ==console tool== 에서 실행하고 결과를 확인할 수 있습니다.
7. 사용자에게 간단한 텍스트 입력값을 받기 위해 ==prompt== 함수를 사용합니다. 
8. 사용자에게 알림 창을 표시할 때는 ==alert== 함수를 사용합니다.
9. 웹 브라우저 창에 내용을 출력하는 프로그램은 ==document.write== 함수를 사용한 것입니다. 
10.  ==comment== 은(는) JS 소스에 설명을 남겨 두기 위한 것입니다.
